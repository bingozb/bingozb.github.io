phantomjs实现服务端屏幕截图

# 前言

前阵子遇到一个需求，微信小程序的某一个界面，需要屏幕截图，然后分享出去。但小程序官方没有开放可以截取超过屏幕的完整界面的 API，也就是只能截取当前屏幕可以看到的内容。而我们需要截长图，所以现有的 API 不能满足需求，只能服务端想办法生成图片。我偶然发现了 phantomjs，可以由服务端去做类似爬虫的操作来保存图片。
